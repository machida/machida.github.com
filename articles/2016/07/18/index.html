Rails で静的ページを作る簡単な方法

これから Rails で作る予定の Web アプリや、すでに出来てる Rails 製の Web アプリに新機能を追加するときなんかに、デザイナだけで「見た目」と「動き」だけがわかる、特にプログラムが入っていない静的な HTML を作ってプロトタイピングしたいときがあります。始めにこういうのを作って、それをチームで共有して議論して作るものが決まったら、エンジニアもデザイナもがっつり手を動かすって流れ。

例えば、

  • Adobe Experience Design CC
  • in vison

などのプロトタイピングツールを使って、プロトタイピングする場合もあるんですが、今しているのはアクションボタンの位置と画面遷移の確認だけでなくて、その後のもっと具体的な、このボタンをクリックすると、モーダルがこんな風に開いて、とか、この数字がこんなアニメーションしながら増減する、とか、実際のデザインに近いものを確認するときの話。デザイナがチームに「こんな感じでいきたいけど、どう？」っていうのを確認する段階。

以前は Assets の中身をそのまま持ってくれば Rails でも使えるので Middleman を使って静的な HTML のプロトタイプを作ってたのですが、よく考えたら最初っから Rails でHTML のプロトタイプを作っても手間は全然変わらないことに気付き、ここ一年くらいは Rails でHTML のプロトタイプを作ってます。「XXX という条件で絞り込んだら結果が XX件でる」みたいなものを画面で出したいときなんかは、もう静的なものではなくて DB を使って seed にデータを入れてしまったほうが楽だったりもするし。

ただ、Middleman は routes ファイルに何も書かなくても、ファイルを置けば静的ページが作られるってところが便利で、Rails でもそういう機能が欲しいって思うときがあります。

その機能を実現するのが thoughtbot が作った「high_voltage」っていう Gem。

thoughtbot は他にも bourbon や paperclip 、factory_girl など、デザイナでもよく聞く有名な Gem を作ってます。

high_voltage のインストール

まずは Rails アプリの Gemfile に、

1
gem 'high_voltage'

を追加して、

1
$ bundle

コマンドを打ちます。

これで high_voltage が使えるようになりました。

ページの作成

app/views のディレクトリの中に pages というディレクトリを作ります。

コマンドラインで作る場合はこう。

1
$ mkdir app/views/pages

で、試しに about ってページを作ってみましょう。

app/views/pages/ のディレクトリの中に about.html.erb（もしくは、about.html.haml、about.html.slim など）を作ります。

コマンドラインで作る場合はこう。

1
$ touch app/views/pages/about.html.erb

適当に about.html.erb （もしくは、about.html.haml、about.html.slim など）に何か書いておきます。

作成したページの確認

Rails を立ちあげて、

1
$ rails s

作成したページにアクセス（http://localhost:3000/pages/about）。

確認できました。

作成したページへリンクを貼る

作成したページのパスは page_path('about') となります。なのでリンクは、

erb

1
<%= link_to 'about', page_path('about') %>

haml、slim

1
= link_to 'about', page_path('about')

と、書きます。

作成したページの URL を変える

今のままだと http://localhost:3000/pages/about という URL になってますが、この pages がいらない！ http://localhost:3000/about がいい、ってこともあります。

その場合、config/initializers/ ディレクトリの中に high_voltage.rb というファイルを作成して、そこに

1
2
3
HighVoltage.configure do |config|
  config.route_drawer = HighVoltage::RouteDrawers::Root
end

と、書きます。

それから Rails を再起動すると、http://localhost:3000/about に URL が変わります。
